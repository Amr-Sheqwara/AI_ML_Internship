# AI/ML Program: Day 1 Environment Setup

## Overview
This repository contains my foundational Python environment setup and my first interactive Jupyter Notebook. It demonstrates the ability to create isolated, reproducible development environments which are critical for professional machine learning workflows.

## Repository Contents
* `Project.ipynb`: A Jupyter Notebook verifying the installation and successful import of core data science libraries (NumPy, Pandas, Matplotlib).
* `requirements.txt`: A frozen dependency tree capturing the exact versions of all installed packages.
