# Day 4: Pandas: Tabular Data

## Overview
This module transitions from the mathematical engine of NumPy to Pandas, establishing fluency in the industry-standard library for real-world, tabular data manipulation and cleaning.

## What I Built
Completed tasks demonstrating the following skills:
* **Data Ingestion & Inspection:** Loaded real-world tabular data (CSV) into structured 2D DataFrames and programmatically evaluated their dimensions, underlying data types, and null-value distributions.
* **Modern Data Cleaning:** Handled missing dataset values (`NaN`) by dropping skewed columns and applying mathematical mean imputation to continuous variables.
* **Boolean Subsetting:** Applied conditional logic directly to DataFrames to filter and isolate specific demographic rows (e.g., adult passengers) without iterating, preparing clean subsets for downstream analysis.
* **GroupBy Aggregation:** Implemented method chaining to segment datasets by specific categories and automatically calculate aggregate statistical summaries (such as demographic counts and mathematical means) in a single optimized operation.
