# Day 5: Matplotlib & Week 1 Mini-Notebook

## Overview
This module serves as the capstone for Week 1, integrating data manipulation with Pandas, numerical operations with NumPy, and data visualization with Matplotlib to analyze and visualize the real-world Titanic dataset.

## What I Built
Completed tasks demonstrating the following skills:
* **Data Ingestion & Cleaning:** Loaded the dataset and handled missing values using Pandas to ensure clean data for downstream visualization.
* **Statistical Analysis:** Applied NumPy functions to calculate summary statistics and generate boolean masks for complex passenger filtering.
* **Data Visualization:** Built fully labeled and customized line plots, scatter plots, bar charts, and histograms to uncover demographic trends and data distributions.
* **Composite Dashboards:** Utilized Matplotlib's subplot functionality to combine multiple distinct visualizations into a single, cohesive figure layout.
