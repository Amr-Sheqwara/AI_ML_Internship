# Day 2: Python Fundamentals & Data Structures

## Overview
This module focuses on establishing fluency in core Python concepts before diving into heavy data science libraries.

## What I Built
Completed Tasks that demonstrating the following skills:
* **Statistical Extraction:** Wrote a custom function to calculate and return data statistics (min, max, mean) as a structured dictionary.
* **Data Filtering:** Refactored multi-line loops into single-line list comprehensions to efficiently isolate specific integers in a dataset.
* **Object-Oriented Design:** Built a custom data class from scratch, demonstrating how to initialize attributes and bind mathematical methods together.
