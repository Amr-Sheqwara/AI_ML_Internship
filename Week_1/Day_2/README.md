
# Day 2: Python for Data Science Fundamentals

## Overview 
 This module focuses on establishing fluency in core Python concepts before diving into heavy data science libraries.

## Learning Objectives

* Use Python's core data types and control flow fluently.

* Write clean, reusable functions with clear parameters.

* Write list comprehensions and read basic object-oriented code

  

## Key Topics

* Core data types: int, float, str, bool, list, tuple, dict, set

* Control flow: if/elif/else, for, while

* Functions: parameters, return values, docstrings

* List comprehensions

* Object-oriented programming basics: classes, attributes, methods

  

## Hands-On Lab: Python Fundamentals Drill

A Jupyter Notebook documenting the practical application of today's concepts:

1.  **Statistical Function:** Built a function returning the min, max, and mean of a list as a dictionary.

2.  **List Comprehension:** Rewrote a standard loop into a single-line comprehension to filter even numbers from a dataset.

3.  **OOP Implementation:** Designed a class demonstrating attribute initialization and method execution for spatial distance calculations.

  

## Tools Used

* Python 3.10+

* Jupyter Notebook
