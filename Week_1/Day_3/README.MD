# Day 3: NumPy: Numerical Computing

## Overview
This module focuses on establishing fluency in core NumPy concepts the first data science library in this Internship.

## What I Built
Completed Tasks that demonstrating the following skills:
* **Matrix Initialization & Inspection:** Generated structured 2D arrays (Matrix) and programmatically verified their dimensional shapes and underlying data types.
* **Advanced Data Slicing:** Extracted specific vectors (isolated columns and rows) from a multidimensional matrix using structural index slicing.
* **Boolean Masking:** Applied conditional logic directly to Matrixes to filter and isolate specific data points without iterating, significantly optimizing extraction times.
* **Array Broadcasting:** Implemented mathematical operations between arrays of different dimensions, utilizing NumPy's broadcasting capabilities to automatically align and calculate values without manual loops.
